클로드 앱으로 시작하기

클로드 앱 스킬을 사용해 퍼블로그 스타일 HTML을 만드는 방법이에요.

시작 전 확인

클로드 앱에서 새 대화를 시작하세요
Chat 또는 Cowork 모두 사용 가능해요.

사용 방법

1
채팅창 진입하여 스킬 시작
방법 A 채팅창 하단 "+" → 스킬 목록에서 'publog-design-system' 선택
방법 B 채팅창에 "퍼블로그 디자인 시스템" 입력 → 트리거로 자동 활성화
2
원하는 페이지·레이아웃 디자인 작업
3
HTML 결과 확인 → 수정 반복
4
채팅창에 "작업끝" 입력
HTML 파일이 제공되고, 웹팀 확인 요청 마무리 단계로 진입해요.
5
노션에 업로드 → 웹팀에 확인 요청
노션 바로가기

클로드 디자인으로 시작하기

클로드 디자인에서 퍼블로그 디자인 시스템을 선택해 작업하는 방법이에요.

시작 전 확인

클로드 디자인에 접속하세요
브라우저에서 claude.ai/design 에 접속해요.

사용 방법

1
디자인 시스템 선택창에서 "퍼블로그 디자인 시스템(표준)" 선택
2
원하는 페이지·레이아웃 디자인 작업
3
완성 후 HTML 다운로드
4
노션에 업로드 → 웹팀에 수정 요청
노션 바로가기
